Сценарии использования:

    1) Поиск информации о спорсмене и команде в которой он состоит

        Главная страница -> страница спортсменов -> страница нужного спорсмена ->
            -> страница команды, в которой он состоит

    2) Поиск информации об участниках конкретного соревнования с известной датой 
        проведения

        Главная страница -> страница соревнований -> страница соревнований с 
            заданной датой проведения -> страница информации о соревновании ->
            -> страница участника соревнования

    3) Редактирование информации о команде, у которой сменился тренер

        Главная страница -> страница команд -> страница конкретной команды ->
            -> страница редактирования информации о команде -> редактирование 
            и сохранение информации о новом тренере

    4) Добавление соревнования со спортом, отсутствующим в базе данных

        Главная страница -> страница видов спорта -> страница добавления вида 
            спорта -> страница соревнований -> страница добавления нового 
            соревнования -> страница нового соревнования -> добавление 
            участников соревнования

    5) Редактирование информации о спортсмене, который перешел в другую команду

        Главная страница -> страница спортсменов -> страница спортсмена ->
            -> страница редактирования информации о спортсмене и изменение 
            текущей команды -> добавление участия спортсмена в предыдущей
            команде до текущего момента
